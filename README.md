# Desk-wheel
The repository for the desk wheel, the best way to store and accesse easily all your desk accessories and decorate at the same time

Made for the thingondesk from hackclub   
Me and my friend (Josephine) wanted to create a way to store and decorate at the same time, so we tought about a ferris wheel, it's kinda funny to have all your paper clips and snacks and access them by turning a potentiometer  
  
We started by sketching everything, the design an the electronics    
Then we put all the electronics together (at first we wanted to use buttons but realised how it would not be easy to use, each buttons needs an item associated to it, you have to learn every button or if you write it, it can change, so we switched to a potentiometer.   
While I was taking care of the electronics, Josephine started on building the structure out of cardboard, I then joined her to decorate it all and put everything together.    
   
   
Here is everything we used :    
-Potentiometer   
-ESP32   
-Servo motor   
-Cardboard   
-Jumpers wires
